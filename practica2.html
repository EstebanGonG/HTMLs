Preguntas

¿Qué valor debe de ser colocado en el atributo for? se utiliza para determinar el ID del elemento de entrada al que se debe asociar el título

¿Cuál es la funcionalidad del atributo required = true? define que es obligatorio llenar ese campo

¿En que consiste el método GET y POST? GET: método para obtener soluciones a nuestras peticiones.POST: método que envía datos a un recurso para que los modifiquen.

¿Cual es la funcionalidad de fieldset y legend? La etiqueta FIELDSET sirve para agrupar los elementos. LEGEND sirve para nombrar o etiquetar un grupo creado con FIELDSET.

¿Qué significan los metódos action, method y target? indican, respectivamente, la ubicación del programa responsable del procesamiento de los datos del formulario y el método usado para enviar la información. El atributo target en form hará que los resultados sean mostrados en una nueva ventana/pestaña.
