Preguntas

Con respecto a la figura anterior definir ¿cual es la utilidad de las etiquetas b, strong, i, em, small, mark , del , ins , sub y sup?

Respuesta:

la etiqueta b es para poner el texto en negritas

la etiqueta strong es para resaltar una parte de texto en negritas

la etiqueta i es para poner el texto en cursiva

la etiqueta em es para marcar con énfasis las partes importantes de un texto

la etiqueta small es para hacer el texto mas pequeño

la etiqueta mark es para marcar resaltando el texto

la etiqueta del es para tachar el texto

la etiqueta ins es para subrayar el texto

la etiqueta sub es para poner subindices

la etiqueta sup es para poner superindices
